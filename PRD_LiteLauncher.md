# LiteLauncher 产品需求文档（PRD）

## 1. 产品目标

- 产品名称：LiteLauncher
- 当前目标平台：Windows 11
- 技术方案：Electron + TypeScript + SQLite
- 产品定位：轻量、快速、可扩展的桌面启动器
- MVP 目标：完成“一键唤起 -> 输入 -> 检索 -> 执行”的闭环

## 2. 目标用户与场景

### 2.1 目标用户

- 办公用户：频繁打开应用、文件夹、文档、网页
- 开发用户：频繁打开项目目录、工具链、命令入口

### 2.2 典型场景

1. 按 `Alt+Space` 唤起，输入 `ch` 打开 Chrome
2. 输入目录关键词，快速打开常用文件夹
3. 输入 `g 关键词` 进行网页搜索
4. 输入 `calc 1+2*3` 计算并复制结果
5. 输入 `clip` 查看并复用剪贴板历史

## 3. MVP 功能范围

### 3.1 唤起与窗口行为

- 默认全局快捷键：`Alt+Space`
- 快捷键冲突时自动回退：`Ctrl+Space`、`Alt+Shift+Space`、`Ctrl+Alt+Space`
- 再次触发快捷键时切换显示/隐藏
- 窗口形态：无边框、居中、置顶、隐藏任务栏
- 系统托盘：显示托盘图标，支持右键菜单（显示主页面、退出）
- 键盘交互：
  - `Enter` 执行当前选中项
  - `Esc` 先清空输入，再隐藏窗口
  - `Up/Down` 在候选列表中切换
  - `Left/Right` 在图标网格中切换

### 3.2 搜索与候选列表

- 输入框支持中文/英文/数字/符号
- 空输入显示首页分区：
  - 最近访问（前 20 条）
  - 置顶（前 20 条）
  - 插件（前 20 条）
- 输入后显示匹配结果（前 20 条）
- 搜索模式展示为图标网格（每行 10 个）
- 搜索结果支持右键置顶/取消置顶
- 候选项展示信息：图标、标题、副标题、类型
- 支持的候选类型：应用、文件夹、文件、网页、命令

### 3.3 索引与数据来源

#### 3.3.1 已实现（MVP 当前版本）

- 开始菜单应用索引：
  - `%APPDATA%\\Microsoft\\Windows\\Start Menu\\Programs`
  - `%PROGRAMDATA%\\Microsoft\\Windows\\Start Menu\\Programs`
- 内置命令项索引：`calc`、`clip`、`settings`、`exit`

#### 3.3.2 规划中（MVP 后续）

- 用户自定义目录的文件/文件夹索引
- 默认排除目录：`node_modules`、`.git`、`dist`、`target`
- 可手动触发索引重建

### 3.4 动作执行

- 应用：按系统默认方式启动
- 文件/文件夹：按系统默认方式打开
- 网页：浏览器打开 URL
- 命令：
  - `calc <表达式>`：计算并复制结果
  - `clip`：打开剪贴板面板
  - `settings`：打开设置面板（可配置显示数量）
  - `exit`：退出应用

### 3.5 剪贴板历史（文本）

- 持续监听系统剪贴板（仅文本）
- 默认保存最近 50 条
- 支持检索、复制、删除单条、清空全部

### 3.6 设置页（当前状态）

- 已实现：
  - 搜索显示数量配置（最近访问/置顶/插件/搜索结果）
  - 配置持久化（重启后保留）
- 待实现：
  - 快捷键配置与冲突检测
  - 开机自启
  - 索引目录与排除规则管理
  - 索引重建
  - 其他行为配置（执行后隐藏、窗口置顶等）

## 4. 搜索与排序规则

### 4.1 匹配策略

- 前缀匹配
- 子串匹配
- 模糊匹配

### 4.2 排序策略

综合评分：

```text
score = matchScore * 0.7 + usageScore * 0.2 + recencyScore * 0.1
```

## 5. 非功能要求

### 5.1 性能

- 唤起耗时目标：<= 150ms
- 输入后结果刷新目标（前 20 条）：<= 50ms
- 动作执行目标：<= 300ms（依赖系统环境）

### 5.2 稳定性

- 崩溃后不丢配置
- 索引可重建
- 关键行为可观测（日志、状态提示）

### 5.3 安全

- 渲染进程禁用 Node 能力
- 系统能力仅由主进程执行
- 预加载层仅暴露最小 API

## 7. 里程碑

### 里程碑一：启动器骨架

- 工程初始化
- 窗口与全局快捷键
- 输入与候选列表

### 里程碑二：搜索与应用索引

- 应用索引
- 搜索排序
- 动作执行与使用记录

### 里程碑三：文件索引与设置页

- 目录索引
- 设置管理
- 索引重建

### 里程碑四：命令与剪贴板

- 命令路由
- 剪贴板历史

## 8. 验收标准（MVP）

- 快捷键可稳定唤起与隐藏
- 输入后可实时显示候选结果
- `Enter/Esc/Up/Down` 行为符合预期
- 可执行应用、文件、文件夹、网页与内置命令
- 搜索结果右键置顶后，空输入“置顶”分区可见且重启后保留
- 剪贴板历史可检索与管理
- 数据可持久化并在重启后保留
