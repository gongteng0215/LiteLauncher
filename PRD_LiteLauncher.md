# LiteLauncher 产品需求文档（PRD）

更新时间：2026-02-26

## 1. 产品目标

- 产品名称：LiteLauncher
- 目标平台：Windows 11（主）、macOS（功能与打包模板支持，未签名）
- 技术栈：Electron + TypeScript + SQLite
- 产品定位：轻量、快速、可扩展的桌面启动器
- 核心目标：形成“唤起 -> 输入 -> 检索 -> 执行”的稳定闭环

## 2. 目标用户与场景

### 2.1 目标用户

- 办公用户：频繁打开应用、文档、网页
- 开发用户：频繁切换工具、目录、命令入口
- 个人效率用户：需要统一入口管理常用动作

### 2.2 典型场景

1. 按 `Alt+Space` 唤起，输入应用名并打开。
2. 输入 `g 关键词` 快速网页搜索。
3. 输入 `calc 表达式` 快速计算并复制。
4. 输入 `clip` 搜索并复用剪贴板历史。
5. 输入 `cash` 进入现金流游戏并推进回合。

## 3. 功能范围

### 3.1 已实现（当前版本）

#### 3.1.1 唤起与窗口

- 全局快捷键：默认 `Alt+Space`
- 快捷键冲突自动回退：`Ctrl+Space`、`Alt+Shift+Space`、`Ctrl+Alt+Space`
- 再次触发切换显示/隐藏
- 窗口：无边框、居中、置顶、隐藏任务栏
- 托盘：显示主页面、退出

#### 3.1.2 搜索与执行

- 搜索类型：应用、文件夹、文件、网页、命令
- 空输入分区：最近访问 / 置顶 / 插件
- 非空输入分区：搜索结果
- 图标网格展示（搜索模式）
- 右键置顶/取消置顶（持久化）
- 动态查询：
  - `g 关键词` 生成网页搜索项
  - `calc 表达式` 生成计算项

#### 3.1.3 内置与插件能力

- 内置命令：`clip`、`settings`、`exit`
- 插件：密码生成器、Cashflow Lite
- Cashflow 当前支持：AI 对战入口、统计入口、职业开局、报表、资产/债务、胜负与存档恢复

#### 3.1.4 剪贴板历史

- 文本监听与持久化
- 默认保留最近 50 条
- 搜索、复制、删除、清空

#### 3.1.5 设置页

- 搜索显示数量配置（最近/置顶/插件/搜索结果）
- 开机启动开关（Windows/macOS，保存后生效）

#### 3.1.6 打包与发布（基础能力）

- `electron-builder` 本地打包脚本：Windows（NSIS/Portable/zip）、macOS（dmg/zip）
- 产物输出到 `release/`
- GitHub Actions：`v*` tag 触发 Windows/macOS 构建并自动上传 GitHub Release 资产（macOS x64 在 `macos-14` 构建，目标 `macOS 12`）

### 3.2 规划中

#### 3.2.1 启动器核心增强（候选）

- 快捷键自定义（唤起键、插件快捷键、全局动作）
- 搜索源开关与索引管理（包含目录、排除目录、手动重建索引）
- 执行后行为配置（保持窗口 / 隐藏 / 复制结果后自动隐藏）
- 命令别名与自定义命令（固定网址、脚本、参数模板）
- 文件快速预览（文本 / 图片）
- 最近文档智能分组（按应用、项目目录、时间段）
- 插件管理页（启用/禁用、排序、配置）
- 插件 SDK（标准命令协议 + 面板协议）
- 托盘菜单增强（快速打开插件/最近命令）
- 搜索结果动作菜单（打开、管理员运行、打开位置、复制路径）
- 设置同步（置顶项、设置、常用命令；后续可选云同步）

#### 3.2.2 分发与产品化增强（候选）

- 自动更新闭环（检查更新、下载进度、重启安装）端到端联调（Windows / macOS）
- Windows/macOS 正式图标资源（合法 `.ico` / `.icns`）
- macOS 签名与公证
- GitHub Release 自动生成更新说明（按提交/标签汇总）
- 崩溃日志与诊断导出（便于用户反馈）
- 匿名使用统计（本地可关闭，用于排序与默认配置优化）

#### 3.2.3 体验扩展（候选，第二批）

- 搜索结果二级动作面板（如 `Tab` 展开动作）
- 截图/OCR 文本提取并纳入搜索
- 工作流宏命令（多步动作串联）
- 快速便签 / Scratchpad 插件
- 项目工作区模式（项目级常用目录/命令/网页）
- 配置导入导出（本地备份/迁移）
- 多显示器布局偏好记忆（按显示器恢复窗口位置/尺寸）
- 国际化语言切换（中文/英文）

#### 3.2.4 智能与开发体验增强（候选，第三批）

- 搜索学习模式（根据点击习惯调整排序，支持重置学习数据）
- 敏感剪贴板过滤（密码/验证码等内容不入历史，支持规则）
- 结果收藏夹分组（常用项按文件夹/场景组织，不仅限置顶）
- 命令参数表单化（复杂命令通过表单输入参数后执行）
- 本地脚本运行沙箱（限制可执行目录与白名单）
- 插件热重载（开发模式下减少重启成本）
- 使用数据看板（本地：常用命令、命中率、耗时等）

#### 3.2.5 主产品使用深度增强（候选，第四批）

- 窗口切换模式（搜索当前已打开窗口并直接切换）
- 浏览器历史/书签搜索（Chrome / Edge 等）
- 剪贴板图片/文件历史（不限于文本）
- 文本片段/模板（Snippets）快速插入
- 文档全文搜索（不仅文件名，支持内容检索）
- 应用二级动作（如管理员运行、打开所在位置、终端打开目录）
- 多配置档（Profile：工作 / 私人）
- 主题系统（颜色、密度、字体等）
- 无障碍模式（大字号、高对比、焦点强化）
- 内置反馈入口（一键导出日志 + 截图 + 描述）
- 发布通道（稳定版 / Beta / Nightly）
- 安全模式启动（插件异常时仅启核心功能）

#### 3.2.6 插件生态与工程韧性（候选，第四批）

- 插件权限模型（文件系统 / 网络 / 执行命令分级授权）
- 插件本地仓库（Manifest Registry，本地安装/卸载/版本管理）
- 插件调试面板（日志、性能、错误堆栈）
- 插件模板生成器（命令型 / 面板型插件骨架）
- 插件 API 版本兼容层（降低升级破坏）
- Feature Flag 系统（新功能灰度开关）
- 配置版本迁移器（升级时自动迁移旧配置）
- 自动更新回滚（更新异常时退回上一版本）
- 运行健康诊断页（数据库/索引/插件/错误状态）
- CI 冒烟测试（打包后自动启动并检查窗口）

## 4. 搜索与排序规则

### 4.1 匹配策略

- 前缀匹配
- 子串匹配
- 模糊匹配
- 拼音首字母与缩写容错

### 4.2 排序策略

```text
score = matchScore * 0.7 + usageScore * 0.2 + recencyScore * 0.1
```

## 5. 非功能要求

### 5.1 性能

- 唤起目标：<= 150ms
- 搜索刷新（前 20 条）目标：<= 50ms
- 执行动作目标：<= 300ms（依赖系统）

### 5.2 稳定性

- 崩溃后不丢配置
- 关键数据可恢复（包括 Cashflow 对局）
- 窗口模式切换稳定（Cashflow 大面板与搜索紧凑面板可互相回退）

### 5.3 安全

- Renderer 禁用 Node 能力
- 系统能力通过 Main 进程执行
- 预加载层暴露最小 API

## 7. 验收标准（当前版本）

- 快捷键可稳定唤起与隐藏
- 搜索可稳定返回并执行目标
- 置顶项重启后仍保留
- 剪贴板历史可管理
- 设置项保存后可生效（含开机启动）
- Cashflow 可玩闭环完整（开局、回合、决策、胜负、恢复）
- Windows 可产出安装包与便携包
- GitHub Actions 可基于 `v*` tag 自动构建并上传 Release 资产（受 GitHub runner 队列影响）

## 8. 版本路线

1. v1.x：补齐设置能力（快捷键/索引）、完善 Cashflow 复盘与 AI 差异化、完成自动更新联调。
2. v2.x：插件生态扩展、签名/公证与分发体验完善。
