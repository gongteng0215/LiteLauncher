# Cashflow MVP 回归清单

更新时间：2026-02-11

## 1. 开局与职业
1. 输入 `cash new cleaner`，应以清洁工开局。
2. 在现金流面板切换职业并点击“以该职业新开”，应按选中职业重开。
3. 面板应显示税后工资、生活支出、债务还款、剩余债务。

## 2. 回合推进
1. 连续点击 3 次“下一回合”。
2. 状态栏应展示结算明细（税后工资/被动收入/总支出/事件/现金）。
3. 日志区应出现“回合结算”与事件记录。

## 3. 买入决策
1. 现金充足时点击“现金买入”，资产与被动收入应增加。
2. 现金不足时应出现“资金缺口”提示。
3. 现金不足时点击“贷款买入”，应新增债务并增加债务还款。

## 4. 失败路径
1. 通过高负债/高支出让现金跌至 `<= 0` 且月净现金流为负。
2. 应出现“现金流断裂”提示并禁用继续推进操作。
3. 打开面板时若恢复到失败存档，应自动新开一局。

## 5. 胜利路径
1. 通过多次买入使被动收入 `>=` 总支出（生活支出 + 债务还款）。
2. 应显示“已达成财务自由”状态。
3. 达成后“下一回合/买入/跳过”应不可继续执行。

## 6. 持久化
1. 在任意进度关闭应用并重新启动。
2. 再次进入 `cash`，应恢复到最近进行中对局（若是失败态应自动重开）。
