# Cashflow Lite 插件产品需求文档（PRD）

更新时间：2026-02-25
适用版本：LiteLauncher `v1.0.4+`

## 1. 产品定位

- 名称：Cashflow Lite（富爸爸现金流插件）
- 形态：LiteLauncher 内置插件
- 目标：通过回合决策训练用户的现金流思维，而非短期“爽感”
- 单局时长：约 10~30 分钟

## 2. 设计目标

1. 决策有后果：每次买入/贷款/跳过都会改变后续走势。  
2. 财务可解释：回合后必须可在报表中看到收入、支出、资产、负债变化。  
3. 可持续迭代：职业、事件、机会、AI 策略均可配置化扩展。  

## 3. 游戏循环

1. 开局选择职业。  
2. 获得机会卡或触发事件。  
3. 玩家决策：现金买入 / 贷款买入 / 跳过。  
4. 回合结算：税后工资 + 被动收入 - 总支出 + 事件影响。  
5. 更新三报表与关键指标，进入下一回合。  

## 4. 核心系统

### 4.1 职业系统（已实现）

- 当前职业：程序员、产品经理、设计师、运营、医生、清洁工。
- 职业属性：工资、税率、生活支出、初始负债、债务月供、机会权重。
- 作用：决定开局难度、现金流结构与机会倾向。

### 4.2 机会系统（已实现）

- 分层：Small / Medium / Big。
- 基础机会字段：成本、月现金流、描述、类型。
- 可执行动作：
  - `buy`：现金买入
  - `buy-loan`：贷款买入
  - `skip`：跳过机会

### 4.3 Big Deal 系统（已实现）

- 在普通机会之外独立抽取。
- 特点：高成本、高回报、高波动。
- 结算包含上行/下行分支：可能改变现金、债务、支出、被动收入。

### 4.4 事件系统（已实现）

- 事件类型：收入、支出、投资、消费。
- 事件影响：`cashDelta`、`expensesDelta`，部分事件可影响 `passiveIncome` 或 `debt`。

### 4.5 财务系统（已实现）

- 三报表：收入表、支出表、资产负债表。
- 关键指标：
  - 月净现金流
  - 被动收入覆盖率
  - 负债率
  - 现金储备月数

### 4.6 阶段系统（已实现）

- `rat-race`（老鼠赛跑）
- `freedom-phase`（自由阶段）
- 跃迁条件：被动收入覆盖基础支出后进入自由阶段，机会和风险参数随阶段变化。

### 4.7 胜负条件（已实现）

- 胜利：被动收入 >= 总支出。  
- 失败：现金流断裂（现金与净现金流同时恶化）。

### 4.8 AI 系统（部分实现）

- 已实现：`cash ai` 开启 AI 玩家并行推进，展示 AI 卡片与最近决策。
- 未完成：多性格 profile 扩展与策略对比面板。

### 4.9 复盘与教练（规划中）

- `cash review` 当前为入口占位。
- 目标能力：决策时间线、关键转折点、反事实分支、行为画像建议。
- 复盘 V1：按回合展示机会、决策、现金流变化与关键指标变化。
- 决策对比：同局对比“你 vs AI”在关键回合的选择差异。
- 教学模式：对高风险决策给出解释型提示（为什么危险/保守替代方案）。
- 结算总结卡：输出本局评分、失败原因、改进建议。

### 4.10 数据可视化与挑战（规划中）

- 图表面板：净现金流、负债率、被动收入覆盖率、资产构成曲线/占比。
- 每日挑战 / 固定种子局：可复现同一局面，便于分享与对比。
- 成就系统：首胜、低负债通关、高被动收入等里程碑。
- 本地排行榜：最快财务自由、最低负债通关、最高被动收入等榜单。

### 4.11 存档与调参（规划中）

- 多存档槽：手动存档、回档、分支尝试。
- 参数配置面板：职业/事件/机会权重可视化调参（不改代码）。

### 4.12 扩展玩法（候选，第二批）

- 市场周期系统（景气/利率/风险偏好变化影响机会与成本）
- 目标卡/梦想线（目标导向模式，不同终局目标带来不同策略）
- 风险管理机制（保险、应急金策略、突发事件缓冲）
- 挑战分享码（导入/导出固定局面）

### 4.13 扩展玩法（候选，第三批）

- 回合回放模式（逐回合播放资产、负债、现金流变化）
- 难度档位（新手 / 标准 / 硬核，影响事件波动与机会权重）
- 职业成长系统（技能提升影响工资、机会质量与抗风险能力）
- 家庭事件系统（结婚、育儿、医疗、搬家等长期支出链）
- 风险偏好问卷开局（为新手推荐职业与策略倾向）

### 4.14 AI 分析与内容扩展（候选，第四批）

- AI 决策理由展示（解释 AI 为什么买入 / 贷款 / 跳过）
- AI 批量模拟 / 锦标赛（多策略多局对比）
- 平衡性分析工具（职业/事件/机会池胜率热图）
- 场景编辑器（自定义开局现金、债务、职业、机会）
- Mod 支持（职业/事件/机会 JSON 扩展包）
- 教程关卡（分步骤教学现金流、杠杆、风险）
- 沙盒模式（无限回合、可调参数、不计入正式战绩）
- 经济术语词典（指标解释与术语说明）
- 赛季任务 / 周挑战（持续回访目标）
- 对局导出报告（图表 + 关键决策摘要）

## 5. 交互与界面

### 5.1 面板结构（已实现）

- 顶部：状态栏、阶段与进度
- 左侧：机会区、职业区、资产区
- 右侧：AI 区、财务报表区
- 底部：回合记录与操作区

### 5.2 关键交互（已实现）

- `Enter`：推进一回合
- `Esc`：返回主页面
- 从 Cashflow 返回时窗口自动恢复主页面紧凑尺寸

## 6. 命令协议

### 6.1 查询命令

- `cashflow` / `cash` / `cf` / `现金流`：打开面板
- `cash new`：新开一局
- `cash ai`：开启 AI 对战
- `cash stat`：查看统计
- `cash review`：复盘入口（占位）

### 6.2 内部动作

- `open`
- `state`
- `reports`
- `next-turn`
- `buy`
- `buy-loan`
- `skip`
- `reset`
- `ai`
- `stat`
- `review`

## 7. 数据持久化

- 存储：SQLite
- 关键表：
  - `cashflow_games`
  - `cashflow_players`
  - `cashflow_professions`
  - `cashflow_assets`
  - `cashflow_debts`
  - `cashflow_events`
  - `cashflow_decisions`
  - `cashflow_stats`
- 能力：关窗后可恢复进行中对局；失败局支持自动重开。

## 8. 非功能要求

- 回合计算稳定、可重放、可测试
- 面板刷新不阻塞主搜索输入
- 数据异常时可降级为新局，避免插件不可用

## 9. 版本路线

1. v1.x（当前）：可玩闭环 + AI 基础 + 统计入口。  
2. v1.1：AI 多性格 + 参数可视化 + 图表面板。  
3. v1.2：复盘页面、关键决策时间线、结算总结卡。  
4. v1.3：教学模式、每日挑战/固定种子局、成就系统。  
5. v2.x：教练模式深化、反事实模拟、市场周期与目标卡。
