# Cashflow Lite 插件产品需求文档（PRD）

更新时间：2026-03-29
适用版本：LiteLauncher `v1.0.12`

## 1. 产品定位

- 名称：Cashflow Lite（富爸爸现金流插件）
- 形态：LiteLauncher 内置插件
- 目标：通过回合决策训练现金流思维
- 单局时长：约 10~30 分钟

## 2. 设计原则

1. 决策有后果：买入 / 贷款 / 跳过都会影响后续走势。
2. 财务可解释：回合后能在报表看到变化来源。
3. 系统可扩展：职业、机会、事件、AI 策略可继续扩展。

## 3. 核心循环

1. 选择职业开局。
2. 获得机会卡或触发事件。
3. 做出决策（现金买入 / 贷款买入 / 跳过）。
4. 回合结算（税后工资 + 被动收入 - 总支出 + 事件影响）。
5. 更新报表与指标，进入下一回合。

## 4. 已实现系统

### 4.1 职业系统

- 当前职业：程序员、产品经理、设计师、运营、医生、清洁工
- 属性：工资、税率、生活支出、初始负债、债务月供

### 4.2 机会系统

- 分层：Small / Medium / Big
- 动作：`buy`、`buy-loan`、`skip`

### 4.3 财务系统

- 三报表：收入、支出、资产负债
- 核心指标：月净现金流、被动收入覆盖率、负债率、现金储备月数
- 结算口径：税后工资 + 被动收入 - 总支出（含债务月供） + 事件影响

### 4.4 胜负条件

- 胜利：被动收入 >= 总支出
- 失败：现金耗尽且净现金流持续为负

### 4.5 AI 与统计

- `cash ai`：开启 AI 对战基础模式
- `cash stat`：查看累计统计

### 4.6 存档恢复

- 基于 SQLite 持久化
- 重启后可恢复进行中的对局

## 5. 当前待补重点

### 5.1 复盘系统（核心待办）

- `cash review` 当前仍为占位入口
- 目标能力：
  - 决策时间线
  - 关键回合对比
  - 失败原因回放
  - 结算总结卡

### 5.2 计划增强

1. AI 多性格策略与差异对比
2. 图表面板（净现金流 / 负债率 / 资产构成）
3. 结算总结卡（评分与建议）
4. 教学模式与固定种子挑战

## 6. 交互规范

- `Enter`：推进一回合
- `Esc`：返回主搜索
- 进入 Cashflow 使用大窗口预设，退出时恢复搜索窗口尺寸
