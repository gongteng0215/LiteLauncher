# LiteLauncher v1.0.10 Release Notes

Release date: 2026-03-10

## Highlights

- Upgraded the `webtools-timestamp` plugin to a dual-conversion panel:
  - `Unix -> Date` and `Date -> Unix` sections
  - second/millisecond unit switch
  - real-time local clock and Unix timestamp display
  - `Now` action for quick input fill
  - auto-convert behavior while typing
- Improved timestamp command protocol with explicit `unit=s|ms`.
- Added stricter timestamp parsing and normalized date output format (`YYYY-MM-DD HH:mm:ss`).
- Updated project documentation baselines to `v1.0.10`.

## Quality

- TypeScript build passes (`pnpm run build`).

## Notes

- Existing hidden WebTools plugins remain hidden by default until parity and layout checks are complete.
