# LiteLauncher v1.0.11 Release Notes

Release date: 2026-03-20

## Highlights

- Added a real development workflow with `pnpm dev`:
  - TypeScript watch mode
  - automatic Electron main-process restart
  - automatic renderer reload
- Added Windows app alias support for searchable and launchable `WindowsApps` entries such as `Codex`.
- Improved app result quality for Windows alias apps:
  - canonical application result instead of generic command fallback
  - real icon resolution
  - AppsFolder launch path support
- Added a first UI end-to-end smoke test with Playwright:
  - launch window
  - open settings
  - search and open the JSON plugin

## Plugin and UX Improvements

- Improved plugin stability during native file dialogs and download flows so the launcher does not auto-hide mid-action.
- Standardized multiline input behavior:
  - `Enter` inserts a newline
  - `Ctrl+Enter` executes the default action
  - `Esc` returns
- Continued parity work across WebTools plugins, including:
  - API client
  - Markdown preview
  - QR code generator
  - SQL formatter
  - config converter
  - diff viewer
  - UA parser

## Quality

- Added new automated coverage:
  - `test:main-flow`
  - `test:windows-alias`
  - `test:e2e:smoke`
  - `test:regression:full`
- Verified with:
  - `pnpm run typecheck`
  - `pnpm run test:regression:full`
  - `pnpm run check:encoding`

## Notes

- This release focuses on developer workflow, Windows alias reliability, and regression safety while continuing WebTools parity work.
