# LiteLauncher v1.0.12 Release Notes

Release date: 2026-03-29

## Highlights

- Added WebTools HTTP Mock as a default visible plugin entry while keeping the plugin catalog clean with a single primary entry.
- Improved plugin section scalability:
  - removed main-process plugin list truncation
  - renderer now paginates plugin section results so more than 20 plugins stay reachable
- Continued renderer panel modularization by moving HTTP Mock panel apply/render logic into the panel implementation registry.

## HTTP Mock Improvements

- Implemented panel-based editing for method, port, path, status code, content type, and body.
- Added start, status, and stop action support with runtime state feedback.
- Added E2E smoke coverage for the lifecycle flow:
  - start mock service
  - hit mock route
  - verify request count/status
  - stop service

## Documentation and Baseline Sync

- Synchronized project documentation to the v1.0.12 baseline.
- Updated default visible plugin list to 21 items (including `webtools-http-mock`).
- Updated WebTools parity and migration docs to the 20-plugin baseline.

## Quality

- Verified build stability with:
  - `pnpm run build`

## Notes

- This release focuses on plugin discoverability, plugin-list scalability, and keeping renderer architecture maintainable while preserving existing user workflows.
