# WebTools 功能齐平清单

更新时间：2026-03-19
基准项目：`E:\AI\webTools`
当前项目：`E:\AI\LiteLauncher`

## 1. 说明

这份清单不再用“是否已经接入 LiteLauncher”作为完成标准，而是用“是否达到原版 webTools 的核心功能、主要交互和关键反馈”作为标准。

状态定义：

- `完整复刻`：核心功能、主流程交互和主要反馈已基本对齐原版
- `部分复刻`：主流程可用，但布局、实时反馈、辅助交互或边界行为仍有差距
- `基础闭环`：已有入口和最小执行链路，但离原版仍有明显差距
- `待复核`：本轮已补过一版，需要继续逐项和原版对照细节

## 2. 总体结论

1. `19/19` 个 WebTools 插件都已经接入 LiteLauncher。
2. `19/19` 个 WebTools 插件都已经开放到主界面。
3. 当前不能简单说“全部做完”，更准确的说法是：
   - 迁移接入：已完成
   - 对外开放：已完成
   - 功能齐平：持续推进中

## 4. 最近已补的重点

- 增加插件面板注册器映射，减少长 `if` 链。
- 增加插件原生交互防隐藏链路。

- `webtools-diff`：补齐差异统计与高亮可视化。
- `webtools-unit-convert`：修复显示科学计数法问题。
- `webtools-ua`：修正 CPU 字段语义，显示“CPU 架构”。
